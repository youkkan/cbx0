3. Stockage et hébergement

Vos données sont stockées sur Supabase (PostgreSQL), hébergé sur des serveurs AWS situés en eu-west-3 (Paris, France), garantissant une localisation des données au sein de l'Union européenne.

Les mots de passe ne sont jamais stockés en clair — uniquement sous forme de hash bcrypt (coût 12).

7. Partage des données

Vos données ne sont jamais vendues ni cédées à des tiers. Elles peuvent être partagées uniquement avec :

  • Supabase — Hébergeur base de données (UE)
  • Vercel — Hébergeur applicatif (USA, avec garanties adéquates)
  • Resend — Service d'envoi d'emails transactionnels (UE)
  • Autorités compétentes sur demande légale obligatoire

8. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées :

  • Mots de passe hashés bcrypt (coût 12)
  • Authentification par JWT signé (expiration 30 jours)
  • Connexions chiffrées TLS 1.3
  • Row Level Security (RLS) activée sur toutes les tables Supabase
  • Code d'accès admin hashé, non stocké en clair
